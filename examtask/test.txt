1.Для перехода в директорию используйте команду cd, например: cd названиедиректории.



2 . ~ обозначает домашнюю директорию текущего пользователя.

  . представляет текущую директорию.

 .. обозначает родительскую директорию текущей директории.

rm - это команда для удаления файлов и директорий в командной строке.



4.cd - команда для изменения текущей директории.



5.cp - команда для копирования файлов и директорий.



6.mv - команда для перемещения или переименования файлов и директорий.



7.Для переименования папки или файла используйте команду mv старое

название новоеназвание.



8.Для перемещения файла или папки используйте команду mv исходный

путь новыйпуть.



9.Флаг -R (или -r) в командах cp, mv и rm указывает на рекурсивное выполнение операции внутри директории, включая все поддиректории и их содержимое.



10.Флаг -r в команде cp и mv также указывает на рекурсивное выполнение операции внутри директории.



11.Git - это система контроля версий, используемая для управления изменениями в исходном коде и других текстовых файлах.



12.git init - это команда для инициализации нового репозитория Git в текущей директории.



13.Для создания новой ветки в Git используйте команду git branch <названиеветки>.



14.Для переключения на другую ветку в Git используйте команду git checkout <название_ветки>.



15.Слияние (merge) в Git - это процесс объединения изменений из одной ветки в другую, чтобы объединить разные линии разработки.



16.git rm - это команда для удаления файлов из репозитория Git.



17.git revert - это команда, которая создает новый коммит, который отменяет (откатывает) изменения, внесенные определенным коммитом, без удаления самого коммита.\
